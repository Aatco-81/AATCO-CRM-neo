توزيع الفرص حسب الأولوية
🔗 ربط Google Sheets
📋 خطوات الربط:
  1. افتح ملف Google Sheets المُرفق (AATCO_Sheets.xlsx)
  2. من القائمة: Extensions → Apps Script
  3. الصق كود الـ Apps Script المُرفق بالملف
  4. انشر كـ Web App: Deploy → New Deployment → Web App
  5. اختر "Anyone" في خانة "Who has access"
  6. انسخ الرابط الناتج وألصقه في الخانة أعلاه
